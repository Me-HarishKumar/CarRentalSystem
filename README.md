Car Rental System (Java)
A console-based Java application that simulates a car rental agency. This project demonstrates core Object-Oriented Programming (OOP) principles, including class relationships, data encapsulation, and collection management.

Features:
Car Management: Maintain a fleet of cars with unique IDs, brands, models, and daily rates.

Customer Registration: Automatically creates customer profiles during the rental process.

Real-time Availability: Tracks which cars are currently out on rent and which are available.

Dynamic Pricing: Calculates total rental costs based on the car's base price and the number of days rented.

Transactional Logic: Handles the full lifecycle of renting and returning a vehicle, updating the system state instantly.
